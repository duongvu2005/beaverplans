# beaverplans

A minimal weekly planner with three levels. A **project** has a title and a deadline, a
project holds **tasks**, and each task is broken into **subtasks** that you assign to
days of the week. The app arranges all of those subtasks into a day-by-day week view,
where a single day can hold as many subtasks as you put there. You plan by filling in
projects and tasks, then check off subtasks to track your progress through the week.

Built in TypeScript with a focus on control and scalability.

> Early and in active development.

## Getting started

Requires [Node.js](https://nodejs.org) 18+.

```bash
npm install
npm run dev      # dev server with hot reload
```

Other scripts:

```bash
npm test         # run tests (Vitest)
npm run lint     # ESLint
npm run build    # type-check and production build
```

Built with React, TypeScript, and Vite.
